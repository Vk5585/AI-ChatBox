Overview

Chat-style application implemented in Java, showcasing basic AI-assisted response logic and GUI programming.
​
Text-based conversation window with separate areas for chat history and user input.
​
Designed as a learning project to explore AI/chatbot behavior and Java event handling.
​

Features

Scrollable chat area displaying both user messages and bot replies.
​
Input text field with send button (and optional Enter key support) for smooth chatting.
​
Simple response engine that replies based on patterns, rules, or integrated AI APIs, depending on implementation.
​
Lightweight and easy to extend with new intents, responses, or external AI services.
